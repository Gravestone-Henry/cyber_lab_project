#  Cyber Security Lab Project

Projeto prático para estudo de segurança ofensiva e defensiva, focado em automação de varreduras e análise de protocolos.

##  Estrutura do Projeto

- `src/scanners/`: Scripts de varredura de rede e portas.
- `src/analyzer/`: Ferramentas para análise de cabeçalhos e tráfego.
- `src/utils/`: Utilitários de log e formatação.
- `docs/`: Relatórios de vulnerabilidades gerados.
- `data/`: Logs de atividades (ignorado pelo Git).
